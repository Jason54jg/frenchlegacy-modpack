**Hypixel SkyBlock (1.21+)**

**FrenchLegacy** is a **Fabric modpack optimized for Hypixel SkyBlock**, combining popular community mods to improve **performance, interface, and in-game information**.

![hypixel](https://cdn.modrinth.com/data/cached_images/6f54f8b74f5321de9aed0d24b418c13e65f9e741_0.webp)

This pack aims to provide a **smooth, stable, and feature-rich** SkyBlock experience.

- ✔ FPS Optimized
- ✔ SkyBlock Enhancements
- ✔ Improved Interface
- ✔ Better Gameplay Experience

---
# ✨ Main Features

<details>
<summary>⚔️ SkyBlock Enhancements</summary>

* **SkyHanni** — Advanced overlays and detailed SkyBlock information
* **Skyblocker** — Open-source SkyBlock mod with multiple features
* **Firmament** — Modern SkyBlock interface improvements
* **SkyblockTweaks** — Various gameplay tweaks
* **SecretRoutes** — Dungeon routes and secret helpers
* **SkyBlockPv** — Extra SkyBlock utilities and stats
* **Jeff’s Mining Utils** — Mining helpers for Crystal Hollows
* **Athen** — Activity tracking and SkyBlock utilities
* **Devonian** — Additional trackers and tools

</details>
